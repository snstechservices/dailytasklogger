üí° Tips & Best Practices

1. Stay Consistent

Start work session at beginning of day. Log tasks as you complete them.

2. Use Descriptive Names

Write clear task descriptions. Use consistent project names.

3. Set Realistic Goals

Start with achievable daily/weekly goals. Adjust based on your patterns.

4. Enable Reminders

Break reminders prevent burnout. Idle detection catches forgotten sessions.

5. Regular Exports

Export weekly for backup. Keep Excel reports for records.

6. Review Analytics

Check weekly trends. Identify peak productivity times.

7. Manage Holidays

Add public holidays in advance. Helps accurate goal calculations.

üîß Troubleshooting

‚ö†Ô∏è Timer Not Starting
  • Check if browser allows JavaScript
  • Ensure LocalStorage is enabled
  • Try refreshing the page
‚ö†Ô∏è Data Not Saving
  • Check browser storage settings
  • Ensure you're not in Private/Incognito mode
  • Storage might be full (check Settings)
‚ö†Ô∏è Reminders Not Working
  • Grant notification permissions in browser
  • Check browser sound is not muted
  • Verify reminder settings are enabled
‚ö†Ô∏è Excel Export Issues
  • Ensure pop-up blocker allows downloads
  • Check browser download settings
  • Try a different browser if problems persist
‚ö†Ô∏è Charts Not Displaying
  • Ensure you have tracked activities
  • Check date range filter
  • Clear browser cache and reload
